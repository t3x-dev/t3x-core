Design question: How long should change indicators persist?

After the user sees "this frame was added this turn", when does it go back to normal?

A

Until next turn

Change indicators (green/yellow/red) stay visible until the next extraction happens. Then they clear and new changes get highlighted.

Simple, predictable. User always sees what just changed.

B

Fade after 5 seconds

Change indicators animate in, then fade to normal after a few seconds. Clean view returns automatically.

Less visual noise, but user might miss changes if not watching.

C

Until next turn + toggle to review (Recommended)

Indicators persist until next turn. Plus a small "Changes" badge in the panel header (e.g., "+2 ~1 -1") that the user can click to re-highlight all changes from the last turn.

Best of both — always visible when fresh, reviewable on demand.
