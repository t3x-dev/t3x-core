6. Returning User — Session Restore

  • App remembers last project + conversation (cookie/localStorage)
  • Lands directly in working context — no navigation needed
  • Sidebar shows recent conversations + project tree
  • "New Chat" always at top for fresh conversations
  • Canvas accessible per project via sidebar or chat header button
