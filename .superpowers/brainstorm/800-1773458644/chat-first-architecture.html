Key Principle

Chat is Home. Canvas is Control Room.
💬 Chat View
Where you create knowledge — talk to LLM, paste docs, upload files. Extraction panel shows live results.
📊 Canvas View
Where you manage knowledge — see lineage, branch, merge, run assertions, check leaf outputs, monitor runners.
