What makes this powerful:
  • Bidirectional: Edit frame in graph → preview regenerates. See missing coverage → go add the frame.
  • Prompt-driven: Quick-select output type (Tweet, Email, Brief, Custom) or write a free-form prompt. Same knowledge, different outputs.
  • Coverage validation: Shows which frames are reflected in the output and which aren't. Suggests fixes.
  • No context switch: Chat stays visible on the left. You can keep chatting to add more knowledge while previewing.
  • Auto-refresh: Preview regenerates when frames change. No manual "regenerate" needed (but button available for explicit re-roll).
