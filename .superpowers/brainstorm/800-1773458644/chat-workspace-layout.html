Panel States

EXPANDED (default)
Full extraction panel visible. Shows sentences, confidence scores, source provenance, and commit action. User can click any sentence to highlight its source in the chat.
COLLAPSED
Panel folds to a thin rail with just an icon + badge count. Chat area expands to full width. Click rail or keyboard shortcut to expand. Badge pulses when new extractions arrive.
LINEAGE VIEW
Toggle from list to lineage mode. Shows how each sentence evolved across commits: original → modified → current. Word-level diffs inline. Click any version to see source conversation turn.

Interaction Details

  • Highlighted text in chat — purple dashed underline shows which parts of the LLM response were extracted. Hover to see which sentence card it maps to.
  • Click sentence card → scrolls chat to source turn and highlights the exact span.
  • Confidence colors — green (≥0.9), amber (0.7-0.9), red (<0.7). Low confidence sentences are auto-deselected for commit.
  • Graph / YAML toggle — Graph view shows entity-relation structure. YAML shows raw structured output.
  • Previous commits section — shows existing knowledge from this branch, dimmed. Helps user see what's already captured vs what's new.
  • Keyboard: ⌘] to toggle panel. ⌘⏎ to commit.
