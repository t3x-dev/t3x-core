Interaction Design

Auto-scroll Behavior

  • Opens with conversation auto-scrolled to source turn
  • Source text highlighted with colored underline
  • User can freely scroll up/down for context
  • Previous value source also marked (dimmed, red border)
  • Non-source turns shown at 35% opacity

Tab Toggle: Previous / Current

  • Toggle between viewing previous and current value source
  • Switching tab auto-scrolls to that turn
  • Both sources always visible when scrolling, but one is focused
  • For added slots: no "Previous" tab (only current)
  • For removed slots: no "Current" tab (only previous)

Clicking Other Slots

  • Click another slot in YAML → viewer stays open, scrolls to new source
  • Header updates to show new slot name
  • Smooth transition, no close/reopen
  • Esc or × closes the viewer

Cross-conversation Sources

  • If previous and current value come from different conversations, show conversation selector
  • Bottom bar shows which conversation is active
  • "Open full conversation" links to chat page
