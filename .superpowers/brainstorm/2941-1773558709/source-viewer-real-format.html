Hard Principle: Source = Verbatim

Never strip formatting

  • Markdown headers, bold, italic → rendered
  • Bullet lists, numbered lists → rendered
  • Code blocks → rendered with syntax highlight
  • Block quotes → rendered as indented blocks
  • LLM answer structure (headers + sections) → kept intact

Why this matters

  • Users recognize conversations by visual pattern, not just text
  • LLM answers have structure — strip it and it's unrecognizable
  • The formatting IS context (a bullet point ≠ a paragraph)
  • Builds trust: "this is exactly what was said"
