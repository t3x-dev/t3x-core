A. Persistent Right Panel (Recommended)

Source panel is always visible. Click a YAML slot → right panel shows source quote with highlighted text. Changed slots show both old and new source. Inline markers (← T3) on YAML lines hint at provenance.

Pros

  • Source always one click away — no mode switch
  • See YAML + source simultaneously
  • Changed slots get special "Changed" header in source panel

Cons

  • Uses screen width — less room for YAML on small screens

B. Slide-in Panel

Source panel slides in from right when triggered (click slot or "Sources" button). Shows all slot sources for the active frame. Overlays the YAML partially. Dismiss with × or Esc.

Pros

  • Full width for YAML when not reviewing sources
  • Focused source review mode

Cons

  • Covers YAML content — can't see both fully
  • Extra click to open/close

C. Inline Expand

Click any YAML slot → source quote expands inline below the slot, right in the YAML card. Changed slots show change provenance with old value. No separate panel.

Pros

  • Source is right next to the data — zero context switch
  • Change provenance directly visible
  • Works on any screen size

Cons

  • Expands card height — can push content down
  • Multiple expanded slots = long card
